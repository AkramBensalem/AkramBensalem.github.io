# Academic Personal Website - Akram BENSALEM

This is a simple, informative, and SEO-friendly personal website created for hosting on Namecheap (or any static hosting service).

## Structure
- `index.html`: French version of the website (Main page).
- `en.html`: English version of the website.
- `style.css`: Shared stylesheet for both versions.

## How to Host on Namecheap
1. Log in to your Namecheap cPanel.
2. Open the **File Manager**.
3. Navigate to the `public_html` directory.
4. Upload all files from this folder (`index.html`, `en.html`, `style.css`) directly into `public_html`.
5. Your website should now be live at your domain.

## Features
- **Multi-language**: Supports French and English with a toggle at the top.
- **SEO Friendly**: Includes meta tags, descriptions, and semantic HTML to help search engines index your site.
- **Mobile Responsive**: Works well on desktops, tablets, and phones.
- **Clean Design**: Focused on academic readability.
